# 📌 Project Overview

This project demonstrates an **end-to-end Data Analytics workflow** using **Excel, SQL, Python, and Power BI**.

The objective is to transform raw E-Commerce sales data into meaningful business insights through data cleaning, SQL analysis, exploratory data analysis (EDA), and interactive dashboard development.

# 🎯 Project Objectives

- Analyze overall sales performance
- Track monthly sales and profit trends
- Identify top-selling products
- Compare category-wise performance
- Discover business insights using SQL & Python
- Build an interactive Power BI dashboard
- Support data-driven decision making

# 📊 Key Business Insights

- Total Sales exceeded **2 Million**
- Total Profit reached **286K**
- Profit Margin remained around **12.47%**
- November recorded the highest monthly sales
- Technology generated the highest category revenue
- Canon products achieved the highest sales

---

# 💡 Business Recommendations

- Increase inventory before peak sales months.
- Promote high-performing products through targeted marketing.
- Focus investment on the Technology category.
- Optimize discount strategies to improve profitability.
- Monitor low-profit months for cost optimization.
